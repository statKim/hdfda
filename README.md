
# hdfda

<!-- badges: start -->
<!-- badges: end -->

`hdfda` is the R-package for High-Dimensional Functional Data Analysis.
It includes the functional linear regression, linear discriminant analysis and two-sample test for high-dimensional functional data.
  
## Installation

You can install the development version of hdfda like so:

``` r
devtools::install_github("statKim/hdfda")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(hdfda)
```
